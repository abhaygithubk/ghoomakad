# ghoomakad — Travel Website

A fully-designed, responsive travel website with a warm luxury aesthetic.

## Files
- `index.html` — Main HTML page
- `style.css` — All styles and responsive design
- `script.js` — Animations, interactions, and form handling

## Features
- Fixed navigation with scroll effect
- Full-screen hero with Ken Burns animation
- Search bar with destination, date, and travelers
- Destination cards with parallax hover effect
- Experience cards with icon layout
- Animated marquee banner
- Masonry-style photo gallery
- Testimonials section
- Contact form with success message
- Responsive footer with links

## Design
- Color palette: Sandy cream, dark espresso, gold accents, forest green
- Fonts: Playfair Display (headings) + DM Sans (body)
- Fully responsive: mobile, tablet, desktop
- Scroll animations via IntersectionObserver
- Images loaded from Unsplash CDN (internet required)

## How to Use
1. Open `index.html` in a browser
2. Customize colors in the `:root` CSS variables in `style.css`
3. Update destination cards, prices, and content in `index.html`
4. Replace Unsplash image URLs with your own images
5. Connect the contact form to a backend (Formspree, EmailJS, etc.)
